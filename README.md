# Shoes app
Interactive Data Visualization for men's shoes in [nike Spain](http://store.nike.com/es/es_es/pw/hombre-zapatillas/7puZoi3).

## Download Data
```
cd ./data_scraping
npm install
node main.js
```
this will update ./data_scraping/products.json

## Serve front end
On root folder:
```
npm install
npm run serve
```
Graphs are interactive.
